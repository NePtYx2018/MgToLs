ShortUrl es un camuflador de Url el cual dejara
tu url acortado similar a una pagina real
como facebook. Su funcionamiento es simple,
primero lee tu url y la guarda en una variable,
luego de esto lee la url a la que quieres que sea
similar y las junta con un @ en el medio. Asi
de simple. Este proceso se puede llevar a cabo 
manualmente o desde este programa.

Este programa esta hecho con fines educativos e
informativos. No me hago responsable de su mal uso.
Aceptas estos terminos al usar el programa.
