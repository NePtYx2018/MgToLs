Sowo es un programa el cual usa la ingenieria
social como su arma principal. El objetivo es
enviar el word infectado a la victima con una
Url antes ya creada ya sea con un reverce tcp
y/o iplogger y enviarselo, para asi obtener
datos de esa persona.


Este programa esta hecho con fines educativos
e informativos. No nos hacemos responsables
de su mal uso, cada cual es responsable de
este mismo. El usuario queda advertido al
instalar el paquete de herramientas.
