MD5Force esta hecho para forzar los hash mediante
md5sum, es un script sencillo pero eficiente.

Pasos para usar:

1) Abrir MD5Force y pegar el hash md5 a decifrar.

2) Pegar la ruta del diccionario para iniciar el
ataque de diccionario.

3) Esperar hasta que el programa termine de atacar.

Este programa esta hecho con fines educativos e
informativos, no me hago responsable de su mal uso,
ya que el usuario queda advertido desde el momento
en el cual instalo el pack de herramientas.
