SpamTe es una herramienta programada en bash
la cual cumple el rol de hacer spam mediante
telegram. Esto se logra mediante la
automatizacion de el cliente para linux
termux-cli el cual nos permite enviar mensajes
desde la terminal.

  Paso 1: Registrar un numero en telegram-cli.

  Paso 2: Escoger la opcion que deseas usar.

  Paso 3: Empezar el ataque al numero.

Este programa esta hecho con fines educativos
y para ser usado en entornos controlados. No
me hago responsable por ningun uso de esta
herramienta, ya que el usuario es advertido
al instalar el programa principal.
