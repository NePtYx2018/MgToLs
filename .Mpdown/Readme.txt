Para usar a Mpdown se usa un programa llamado
youtube-dl es de uso libre pero de todas formas
creditos a su debido creador. Este programa es
para descargar videos ya sea de YouTube, Facebook,
Twitter, etc.

Â¿Que es el codigo del video?

Es para especificar el tipo de descarga ya sea
mp4, mp3, webm, m4a, etc. Por lo que el codigo
se encuentra en una lista que se entrega al
introducir la url a descargar el video. Este
codigo esta en la parte inferior derecha de la
pantalla, de hecho se puede observar una esquina
dice "Code" por lo que ayi hay que elegir que formato
deseas descargar para luego escribir el codigo.
