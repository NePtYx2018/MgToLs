RemPc (Remote Pc) esta programado para poder tener
el control de un computador mediante termux sin necesidad
de metasploit, eso si necesitas tener un computador con
python instalado y usar pyinstaller para luego ejecutar lo
siguiente en cmd:

pip install pyinstaller

(Luego movemos el archivo creado con RemPc a donde esta
ubicada la ventana cmd)

pyinstaller -w elnombredetubackdoor.py

Luego de esto se puede personalizar el icono como quieran
es super facil y sencillo. Ademas se pueden interesar en la
programacion en python.

Este programa aun esta en fase beta por lo cual funciona
pero no al 100% en unos meses de seguro estara ya profundiazado.

Este programa esta hecho con fines educativos e informativos,
no me hago responsable de su mal uso.
