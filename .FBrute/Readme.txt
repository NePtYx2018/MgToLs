FBrute herramienta para realizar ataques de diccionario
a cuentas de facebook.

Paso 1: Obten el id de la victima, correo o numero
de telefono vinculado a la cuenta de el anteriormente
dicho.

Paso 2: Descarga un diccionario que contenga contraseñas
posibles para la cuenta.

Paso 3: Introduce la ruta de donde se descargo y/o tienes
el diccionario, por ejemplo: 

/sdcard/Diccionario.txt

Paso 4: Esperar a que el programa pruebe todas las contraseñas
posibles.

Este programa esta hecho con fines educativos. Los creadores
no se hacen responsables de su mal o buen uso.
